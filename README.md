# ASNTool
Simple ASN Lookup Tool

```
Get ASN informations from IP Addresses

Usage:
  ASNTool asn [flags]

Examples:
ASNTool asn [IP] [IP] ...

Flags:
  -h, --help   help for asn
```

```
Get IP ranges from ASNs

Usage:
  ASNTool net [flags]

Examples:
ASNTool net [ASN] [ASN] ... (without AS prefix)

Flags:
  -h, --help   help for net
```
